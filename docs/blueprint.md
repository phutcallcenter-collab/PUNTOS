# **App Name**: Call Center Tracker

## Core Features:

- Registro de Incidentes: Registra incidentes (tardanzas, ausencias, errores) con códigos asociados y cálculo automático de puntos basado en el día de la semana y tipo de incidente.
- Control de Vacaciones y Permisos: Registra y controla los días de vacaciones, licencias por enfermedad y otras ausencias para cada representante.
- Gestión de Turnos: Gestiona las asignaciones de turnos, incluyendo cobertura, intercambios de turnos, permisos y días libres.
- Reportes Automatizados: Genera reportes que resumen los incidentes, la asistencia y otros datos relevantes para cada representante.
- Resumen del Agente: Muestra un resumen del rendimiento del agente, incluyendo el número de incidentes, los puntos acumulados y los días de vacaciones/permiso restantes.

## Style Guidelines:

- Color primario: Azul oscuro (#2E3A87), que evoca fiabilidad y profesionalismo.
- Color de fondo: Azul claro (#D9E2EF), proporcionando un espacio de trabajo tranquilo y organizado.
- Color de acento: Naranja (#C95519) para resaltar información y acciones clave.
- Fuente del cuerpo y del encabezado: 'PT Sans', una fuente humanista sans-serif que combina un aspecto moderno y cálido.
- Usa íconos claros y reconocibles para representar diferentes tipos de incidentes y acciones.
- Un diseño limpio y tabular para facilitar la entrada y revisión de datos, optimizado para los flujos de trabajo del centro de llamadas.
- Animaciones sutiles para las actualizaciones de estado y la generación de informes, mejorando la retroalimentación del usuario.
